RoCAS: Rock Color Analysis System - Help Documentation

I. Single Image Analysis

1.1 Loading & Saving

  • Load Image: Import a single rock image via the toolbar or menu.
  • Save: Export the current display image to your local drive.
  • Crop: Supports rectangular ROI (Region of Interest) cropping; press Enter to confirm.
  • Flip/Rotate: Standard image orientation adjustments.

1.2 Image Segmentation

  • Algorithm Selection: Includes GrabCut, Color Thresholding, Edge Detection, Adaptive Thresholding, Watershed, K-means, etc.
  • Multi-selection: Combine algorithms; default settings can be configured in "Settings."
  • Preview: Side-by-side comparison of Original, Segmented, and Mask images.
  • High-DPI Export: Supports 300/600/1200 DPI for high-quality SCI journal publication.

1.3 Color Identification

  • Color Analysis: Intelligent identification using color databases or pre-trained models.
  • Color Prediction: Utilizes DL models for prediction (configure model path in Settings).
  • Visualization: Export 3D color space plots and histograms for research papers.
  • Data Export: Save identification results in professional Excel/CSV formats.

II. Dataset Management

2.1 Loading Datasets

  • Load Folder: Select the root directory containing rock image batches.
  • Auto-Switch: The system automatically switches to the "Dataset" tab upon loading.

2.2 Statistics

  • Summary: Displays total count, format distribution, and subdirectory statistics.
  • Data Interaction: Table content supports right-click copying.

2.3 File Structure

  • Tree View: Visual representation of the dataset's directory hierarchy.

2.4 Image Preview

  • Selection: Ctrl+Click (Multi-select), Shift+Click (Range-select), Ctrl+A (Select All).
  • Double Click: Enlarge image for detailed inspection.
  • Batch Processing: Apply segmentation to all selected images simultaneously.

III. Image Segmentation Workflow

3.1 Single Segmentation

  • Load an image in the "Single Image" tab and click "Image Segmentation."

3.2 Batch Segmentation

  • Full Batch: Run segmentation on all images in a loaded dataset.
  • Selected Batch: Run segmentation only on images highlighted in the preview.

3.3 Algorithm Config

  • Available: GrabCut, Color Threshold, Canny Edge, Adaptive Threshold, Watershed, K-means.
  • Default algorithm logic can be set in Settings → Segmentation Settings.

IV. Machine Learning Pipeline

4.1 Data Preparation

  • Generate Train/Val/Test CSV files and corresponding directories from datasets.
  • Supports auto-labeling based on subfolders or manual annotation.

4.2 Model Training

  • Architectures: ResNet, EfficientNet, MobileNet, and Custom CNN.
  • Parameters: Configure Paths, Epochs, Batch Size, and Early Stopping patience.
  • Directory Persistence: One-click selection of recently used folders.
  • Logging: Real-time training logs synced to the main window and saved by function.

4.3 Model Evaluation

  • Load evaluation reports to view Confusion Matrices, Classification Reports, and Error Samples.

V. Settings & Configuration

5.1 DPI Resolution

  • Choose between 300/600/1200 DPI for professional publication standards.

5.2 Export Preferences

  • Configure Title visibility, Filename formats, and SCI Color Themes.

5.3 Segmentation Logic

  • Set default algorithms and toggle "Reminder" notifications.

5.4 Segmentation Model (Deep Learning)

  • U-Net model path configuration, GPU acceleration toggle, and Training entry.

5.5 Identification Model

  • Classification model paths and toggle switch for model-based recognition.

5.6 Logging & Maintenance

  • Auto-save settings, custom paths, and naming rules (HostName + Date).

5.7 ML Environment

  • TensorBoard integration, port configuration, and log directory management.

VI. Miscellaneous

  • Functional Descriptions: Accessible via Settings or F1 for copyable dialogs.
  • UI Capture: Toolbar screenshot tool for capturing high-definition interface images.
  • Sub-window Support: Individual modules support independent screen captures.
  • Help: F1 or Help → Documentation for browser-based help.
